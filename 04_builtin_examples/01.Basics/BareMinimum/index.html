void setup() {

}

void loop() {
  // put your main code here, to run repeatedly:
}

This sketch simply shows the standard template when coding for any Arduino boards. As you can see, there are typically only 2 functions:

  • setup(): for initialization
  • loop(): normally, a forever loop that runs infinitely

From this bare minimum template, we can easily tell Arduino is a Microcontroller with a single thread, and of course for a single user.
